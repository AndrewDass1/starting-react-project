# To-Do-List Application

This repository contains files that will start a To-Do Application. It is made in JavaScript and the front-end framework React.js. Currently, this application is still being developed.

To replicate this project on your local machine, have the npm package from Node.js (1) installed already.

After installing npm and it is on the local machine, go to an empty folder or directory in the machine. Running running commands in an directory many files will be downloaded from React.js and be placed in the same directory where the command was run. To download the files to start a React.js application, run the following command:

```
npx create-vite@latest --template react .
```

The above command will download the basic files to run a React.js server that will explain how to use React.js with vite. Vite provides more features to build web applications faster (2). The files code in this directory has been modified to display a To-Do app instead of React and Vite documentation.
